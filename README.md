## Run Locally

Clone the project

```bash
  git clone https://github.com/dhatGuy/backdrop-challenge.git
```

Go to the project directory

```bash
  cd backdrop-challenge
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn android
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
```

W/ coverage

```
  yarn test:coverage
```
